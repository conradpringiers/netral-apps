# Stop clicking, start writing
### 🌐 Netral Apps
Netral Apps is a suite of ultra-simple, text-driven productivity tools.
Write once — generate websites, presentations, and more — all using the same unified Netral Syntax.
No complex UI.
No learning curve.
Just write text, and Netral turns it into clean, structured, beautiful output.
## ✨ Features
### Netral Syntax
A minimal, expressive, and unified syntax used across all Netral tools.
Think of it as a lightweight way to create structured content without “builder fatigue”.
- Consistent across all tools
- Easy to learn
- Fast to write
- Human-readable
- Portable
***
## Available tools:
### 📊 Netral Deck — Create Presentations with Text
Turn simple text files into clean, modern slides.
- Markdown-inspired structure
- Themes included
- Fast, distraction-free writing flow
- Perfect for quick ideas, teaching, or lightweight presentations
- Exportable & embeddable
### 🏗️ Netral Block — Build Websites from Text
- Generate responsive static websites using the same syntax.
- Zero HTML/CSS required
- Extremely lightweight output
- Pre-made templates
- Clean layouts
- Ideal for personal sites, documentation, project pages, and micro-sites
## 🎯 Why Netral?
Because most tools are overloaded.
Netral focuses on speed, clarity, and minimalism.
- 100% free
- Works everywhere (web, local, portable)
- Zero-friction workflow
- Ideal for students, developers, educators, indie makers, writers, or anyone who prefers text over UI
